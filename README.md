Chess::Tournaments
==================

A chess tournament tracking application

Setup
=====

1. edit config/initializers/devise.rb
2. setup your apache configuration with ENV variables for:
```
    GITHUB_APP_ID
    GITHUB_APP_SECRET
    TWITTER_APP_ID
    TWITTER_APP_SECRET
    FACEBOOK_APP_ID
    FACEBOOK_APP_SECRET
    SECRET_KEY_BASE
```
3. edit the mailer domain in config/environments/production.rb
